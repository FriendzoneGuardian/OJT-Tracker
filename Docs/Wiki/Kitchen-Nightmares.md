## 🥗 First Time Prep (The Mise en Place)
You can't just throw code in a pan and hope it works! You need to prep your station!

1. **Clone the Repo**: 
   ```bash
   git clone https://github.com/FriendzoneGuardian/OJT-Tracker.git
   ```
   *Do it now! Don't just stand there like a frozen cod!*

2. **ENTER THE FOLDER**:
   ```bash
   cd OJT-Tracker
   ```
   *If you try to install outside the folder, I will personally throw your laptop in the bin!*

3. **THE VIRTUAL ENVIRONMENT**:
   ```bash
   python -m venv venv
   ```
   *It's a VIRTUAL environment, not a virtual reality where you're a good coder! Get it done!*

4. **ACTIVATE & SEASON**:
   - **Windows**: `venv\Scripts\activate`
   - **Mac/Linux**: `source venv/bin/activate`
   *If I don't see that `(venv)` in your terminal, it's **BLAND**!*

5. **INSTALL THE INGREDIENTS**:
   ```bash
   pip install -r requirements.txt
   ```
   *This takes a moment. Use the time to reflect on why your last project was so overcooked!*

6. **START THE SERVICE**:
   ```bash
   python app.py
   ```
   *FINALLY! Service is live at http://localhost:8080. GET IT OUT!*
